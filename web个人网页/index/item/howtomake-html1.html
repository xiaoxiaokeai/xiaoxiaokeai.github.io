HTML document

HTML document = web page. HTML tags are often referred to as HTML tags. The HTML tag is the keyword surrounded by the brackets, and HTML tags usually come in pairs.

Web brower

The role of a Web browser is to read HTML documents and display them as Web pages. Instead of displaying the HTML tag, the browser USES tags to explain the content of the page
